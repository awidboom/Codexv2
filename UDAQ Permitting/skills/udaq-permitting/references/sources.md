# Primary sources (Utah DEQ / Utah Administrative Rules)

Keep links in code spans so they're easy to copy.

## UDAQ laws + rules landing page

- `https://deq.utah.gov/air-quality/air-quality-laws-and-rules`
  - DAQ-curated navigation to rules, statutes, SIP materials, and related resources.
  - Useful starting point, but always confirm rule text on the Utah Administrative Rules site.

## UDAQ home (website navigation hub)

- `https://deq.utah.gov/division-air-quality`
  - DAQ hub page with links to Permitting, Compliance, Laws and Rules, and DEQ's archive document search.

## Utah DEQ public document repository (Laserfiche WebLink)

- `https://lf-public.deq.utah.gov/WebLink/Browse.aspx?id=8436&dbid=0&repo=Public`
  - Browseable folder tree including:
    - `Agency Interest - ...` program folders (general documents)
    - Numeric range folders (used to partition facility IDs)

- Agency Interest - Permitting (guidance, templates, internal process docs published for public access):
  - `https://lf-public.deq.utah.gov/WebLink/Browse.aspx?id=385828&dbid=0&repo=Public`
  - Local mirror (if downloaded into this repo): `data/weblink/agency-interest-permitting/`

## Utah Administrative Rules (official rule text)

- Use the Utah Administrative Rules site linked from the DEQ laws/rules page for:
  - Exact section/subsection citations
  - Effective dates and rule history
  - The current consolidated text

Common entry points (JS-heavy site):

- `https://adminrules.utah.gov/`
- `https://adminrules.utah.gov/public/search//Current%20Rules` (DEQ-linked "Current Rules" search)
